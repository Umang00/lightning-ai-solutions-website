## üêõ Troubleshooting

**Issue: robots.txt not accessible**
- Check Vercel deployment logs
- Ensure file is in `public/` folder
- Wait 2-3 minutes after deployment

**Issue: Sitemap showing 404**
- Verify `app/sitemap.ts` exists
- Check Next.js build output
- Try accessing `/sitemap.xml` directly

**Issue: Structured data not validating**
- Check for JSON syntax errors
- Ensure proper quotes (double quotes, not single)
- Verify @context and @type fields

**Issue: OG image not showing**
- Confirm file is in `public/og-image.jpg`
- Check file size (< 1MB)
- Test with Facebook Debugger
